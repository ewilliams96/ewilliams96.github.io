CryptCoin

Java
CSE 260 course project. Implements a the CryptCoin cryptocurrency protocol, which involves constantly receiving block and transaction data that must be validated while simultaneously using the most recent form of the data to quickly push blocks onto the block chain. To implement this, I came up with data structures to represent the block chain while taking advantage of serialization, and used multiple threads to mine blocks.
